# 📱 PWA (Progressive Web App) Feature

## ✅ Features Implemented

Your portfolio is now a **Progressive Web App (PWA)** with the following capabilities:

### 🎯 Core PWA Features
- **Installable** - Users can install the app on their device (mobile & desktop)
- **Offline Support** - Works offline with cached assets
- **App-like Experience** - Opens as a standalone app without browser UI
- **Auto Updates** - Service worker automatically updates when new version is available

### 🚀 Technical Implementation

#### 1. **Service Worker**
- Auto-generated by `vite-plugin-pwa`
- Caches all static assets (JS, CSS, images, fonts)
- Network-first strategy for dynamic content
- Cache-first strategy for Google Fonts and CDN assets

#### 2. **Web App Manifest**
- Located at `/manifest.json`
- Defines app metadata (name, icons, theme colors)
- Enables "Add to Home Screen" functionality

#### 3. **Install Prompt**
- Custom install prompt component (`PWAInstallPrompt.jsx`)
- Shows after 30 seconds of browsing
- Dismissed prompts won't show again for 7 days
- Smooth animations with Framer Motion

### ⚙️ Configuration

#### Vite PWA Config (`vite.config.js`):
```javascript
VitePWA({
  registerType: 'autoUpdate',  // Auto-update service worker
  manifest: {
    name: 'Awanda - Fullstack JavaScript Developer',
    short_name: 'Awanda Portfolio',
    theme_color: '#667eea',
    background_color: '#0a0e27',
    display: 'standalone',
  },
  workbox: {
    maximumFileSizeToCacheInBytes: 10 * 1024 * 1024, // 10 MB
    // Runtime caching for external resources
  }
})
```

### 🎨 Customization

#### Change Theme Color
Edit `vite.config.js`:
```javascript
manifest: {
  theme_color: '#667eea',  // Change to your brand color
}
```

#### Update Install Prompt Delay
Edit `src/Component/PWAInstallPrompt.jsx`:
```javascript
setTimeout(() => {
  setShowPrompt(true);
}, 30000);  // Change from 30000 (30s) to your preferred delay
```

#### Change Dismiss Duration
Edit `src/Component/PWAInstallPrompt.jsx`:
```javascript
if (now - dismissedAt < 7 * 24 * 60 * 60 * 1000) {  // 7 days
  return;
}
```

### 🔧 Troubleshooting

#### Service Worker Not Registering?
- Ensure you're using HTTPS (or localhost)
- Check browser console for errors
- Clear browser cache and reload

#### Install Prompt Not Showing?
- Must be served over HTTPS
- User must interact with the site first
- Check `beforeinstallprompt` event in DevTools

#### App Not Working Offline?
- Check if all assets are precached
- Verify service worker is active
- Check Cache Storage in DevTools
